# 游伴 Agent

**城市文化旅行中的连续决策旅伴。**

游伴理解你为什么想出发，把模糊愿望变成可走的方案；在天气变化、体力消耗或同行偏好冲突时，主动提出调整并说明理由；站在每一站，帮你建立「为什么值得看、重点看哪里」的理解，而不是堆砌信息。

## 本地体验

双击 `start.command`（macOS）或 `start.bat`（Windows），也可运行：

```sh
npm start
```
